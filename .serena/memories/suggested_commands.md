# Suggested Commands

## Apply Configuration Changes

```bash
darwin-rebuild switch --flake .#Mac-big
```

Note: Requires sudo for system activation.

## Format Nix Files

```bash
nix fmt
```

## Update Flake Inputs

```bash
nix flake update
```

## System Commands (Darwin)

Standard unix commands are available: `git`, `ls`, `cd`, `grep`, `find`
